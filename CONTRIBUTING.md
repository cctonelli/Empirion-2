# Contributing to EMPIRION

Welcome! We are excited that you want to contribute.

## How to Contribute
1.  **Check Issues:** Look for open issues or create a new one.
2.  **Fork & Branch:** Create a feature branch (`feat/your-feature`).
3.  **Code Style:** Follow Clean Code, SOLID principles, and use TypeScript strictly.
4.  **Database:** If you change the schema, create a new migration in `supabase/migrations/`.
5.  **Documentation:** Update the relevant files in `docs/` and the `CHANGELOG.md`.

## Standards
- Use Brazilian Portuguese for comments and internal documentation where requested.
- Ensure all tests pass before submitting a PR.
